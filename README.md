# Embedding Map
Embedding Map for Gemma 2 2B


## Getting Started

Request access to the [Gemma 2 2B model](https://huggingface.co/google/gemma-2-2b) on Hugging Face then run the following commands:

```bash
python3 -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
python main.py
```
